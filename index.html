Project Description

Learning Website is a lightweight educational web application built using Laravel (PHP) with Blade templates, JavaScript, CSS/SCSS, and a frontend build tool chain (Vite). It provides a structured way to host and navigate learning content-lessons, resources, and assets-while organizing routes, views, and database models in a clean and maintainable architecture. Rainfall Prediction

This project applies machine learning techniques to analyze historical weather data and predict future rainfall patterns, aiming to improve accuracy in short-term weather forecasting.

Project Description

This project focuses on building a machine learning-based model to predict rainfall using historical weather data. By analyzing patterns in parameters such as temperature, humidity, and previous rainfall records, the system aims to forecast upcoming weather conditions with improved accuracy. The project not only explores preprocessing and feature selection but also compares different algorithms to evaluate performance and reliability. The ultimate goal is to provide a data-driven approach to weather forecasting that can support planning and decision-making in agriculture, disaster management, and daily life activities.

APPROACH

The approach begins with data collection and cleaning to handle missing values, outliers, and inconsistencies in the dataset. Exploratory data analysis (EDA) is then performed to identify key features influencing rainfall, with visualization techniques like correlation heatmaps and distribution plots. Next, the data is split into training and testing sets, followed by applying supervised learning models such as Logistic Regression, Random Forest, and Decision Trees. The models are tuned using cross-validation and evaluated with metrics like accuracy, precision, recall, and F1-score. Finally, the best-performing model is selected, and the results are validated to ensure generalization. This structured pipeline ensures a robust prediction framework grounded in reproducibility and data-driven insights.

Project Description

This project is a web-based time utility that combines two crucial elements: a Real-Time Clock with Calendar that shows the time and date in real time and a Stopwatch for accurate time monitoring. Completely constructed with HTML, CSS, and JavaScript, it offers a responsive and fluid user experience on all platforms. While the real-time clock continuously updates to keep users in sync with the precise time and date, the stopwatch provides start, pause, and reset functions for precise measurement. With its simple and easy-to-use layout, the tool offers an interactive way to efficiently manage time, combining functionality and aesthetic appeal into one cohesive platform.

APPROACH

This project's methodology involves designing two main modules: a Stopwatch and a Real-Time Clock with Calendar, using HTML for structure, CSS for styling and responsiveness, and JavaScript for dynamic functionality. The Stopwatch uses interval-based calculations and event-driven controls (start, pause, reset) to accurately track time that has passed, while the Clock & Calendar module uses JavaScript's Date object and periodic updates to display real-time hours, minutes, seconds, and dates. Both modules are incorporated into a single, responsive interface with consistent styling to ensure usability across devices, offering a user-friendly and effective web-based time management tool.

Simple As Possible 1

This project implements the SAP-1 (Simple As Possible) 8-bit computer in Proteus, demonstrating core computer functions-processing, control, and memory-using discrete components as a foundation for understanding complex architectures.

Project Description

The project focuses on designing and implementing an SAP-1 (Simple-As-Possible) 8-bit computer using combinational and sequential circuits. It integrates core components such as an 8-bit bus, 16×8 RAM, program counter, memory address register, input unit, arithmetic and logic unit (ALU) with accumulator and B register, control unit, and output register. The system executes a basic instruction set-Load, Add, Subtract, Output, and Halt-driven by a 555 timer-based clock with both auto and manual modes. By simulating fundamental CPU operations in Proteus, this project demonstrates the basic principles of computer architecture and provides a solid foundation for understanding more complex digital systems

APPROACH

First, design and interconnect the essential modules: a 4-bit program counter, instruction register, memory address register (MAR), 16×8 RAM, accumulator and B register, ALU, output register, input unit, and the control sequencer. Next, define the instruction set and the micro-operations for each instruction (LOAD, ADD, SUB, OUT, HALT). Then, implement the control logic (control signals and timing states) in Proteus, synchronized via a clock (e.g. 555 timer or internal clock module). After that, simulate and test each component individually (ALU operations, memory read/write, register transfers) and then integrate them, ensuring proper signal flow and control sequencing. Finally, run full instruction cycles in simulation to verify correct execution of programs, debug timing or signal issues, and optimize for reliability. The whole project was built on Proteus 8 Professional software.
